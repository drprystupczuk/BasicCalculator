Basic Calculator (Academic Project)

This repository contains a simple Windows Forms "Basic Calculator" application developed as an academic exercise. The project demonstrates basic C# and WinForms concepts such as UI controls, input validation, and simple arithmetic operations.

Files included
- The main source is under the "PRYIG CALCULADORA" folder (Windows Forms project).

Notes
- This project is for educational purposes only.

Publishing to GitHub
1. Initialize a local repository if needed: `git init`.
2. Add files and make a first commit:
   - `git add .`
   - `git commit -m "Initial commit: Basic Calculator academic project"`
3. Create a new repository on GitHub (via the website) and follow the instructions to add the remote, for example:
   - `git remote add origin https://github.com/<your-username>/<repo-name>.git`
   - `git push -u origin main` (or `master` depending on your default branch)
