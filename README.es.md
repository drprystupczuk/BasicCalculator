Calculadora Básica (Proyecto Académico)

Este repositorio contiene una aplicación sencilla de Windows Forms llamada "Calculadora Básica" desarrollada como ejercicio académico. El proyecto demuestra conceptos básicos de C# y WinForms como controles de UI, validación de entrada y operaciones aritméticas simples.

Archivos incluidos
- El código fuente principal está en la carpeta "PRYIG CALCULADORA" (proyecto Windows Forms).

Notas
- Este proyecto es únicamente con fines educativos.
